# Smart Bachat - Tech Blog Website

Smart Bachat is a responsive tech blog homepage built using **HTML, CSS, and JavaScript**.
It features a modern responsive navbar, hero section, blog cards, and a clean footer layout.

## 🚀 Features

* Responsive Navigation Bar with Hamburger Menu
* Hero Section
* Blog Card Grid Layout
* Newsletter Subscription Footer
* Mobile Responsive Design
* Clean Modern UI

## 🛠️ Built With

* HTML5
* CSS3
* JavaScript
* Font Awesome
* Google Fonts (Poppins)
